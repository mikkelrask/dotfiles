Quick start

The FAST Quick start​

If you just wanna get going, you can copy/paste the following:

curl -fsSL https://raw.githubusercontent.com/mikkelrask/dotfiles/master/installer.sh | bash

This will automatically download the complete repo, change directory into it, and run the second-stage installer, beforefinally clean up after itself, after doing so.

A bit more controlled, a bit more manual​

Install everything
mkdir -p $HOME/.tmp/dotties
git clone git@github.com:mikkelrask/dotfiles.git $HOME/.tmp/dotties
cd .tmp/dotties
chmod +x install*

and from there decide weather to run install.sh, intall_packages.sh, or an individual package.

Requirements​

Besides the main packages that are documented I also use different utils and quality-of-life improvements to make it all work - most of the requirements are strictly for neovim, and specifically if you use it to write a given language. I.e you don't have to install go, if you never plan on writing anything golang. (in neovim run :checkhealth to verify the languages you need, is present) All documentation and code examples are using doas rather than sudo when needing elevated rights, but if you don't use doas simply change it accordingly.
